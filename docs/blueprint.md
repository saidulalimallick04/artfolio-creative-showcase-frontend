# **App Name**: ArtFolio

## Core Features:

- Landing Page Mosaic: Dynamically display a random selection of user-uploaded images in a visually appealing mosaic/masonry layout.
- User Registration: Enable users to create accounts securely with email verification.
- Secure Authentication: Implement secure login functionality using password hashing and session management.
- Profile Dashboard: Provide a private dashboard where users can upload images and view their uploaded images.
- Public User Profile: Generate public user profiles accessible via /profile/[username] to showcase a user's artwork in a mosaic formation.

## Style Guidelines:

- Primary color: Muted Indigo (#5C6BC0) to evoke creativity and calm.
- Background color: Light Gray (#F5F5F5) to ensure content is readable and prominent.
- Accent color: Warm Amber (#FFB300) to highlight key interactive elements and call-to-actions.
- Headline font: 'Belleza' (sans-serif) for a modern and stylish look in headings.
- Body font: 'Alegreya' (serif) for a readable and elegant experience in descriptions and body text.
- Use simple, outlined icons for a clean and modern feel across the site.
- Implement a responsive grid layout that adapts seamlessly to different screen sizes.
- Use subtle transitions and animations on hover and page transitions to enhance the user experience.
